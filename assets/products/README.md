# assets/products 폴더

캐러셀에서 사용하는 실제 제품 이미지 파일 저장 위치입니다.

## 파일 목록 (index.html carouselSlides 배열 참조)

| 파일명 | 기기 | 상태 |
|--------|------|------|
| `fold8-ultra.jpg` | Galaxy Z Fold8 Ultra | ⏳ 사용자 저장 대기 |
| `fold8.jpg` | Galaxy Z Fold8 | ⏳ 이미지 준비 후 저장 |
| `flip8.jpg` | Galaxy Z Flip8 | ⏳ 이미지 준비 후 저장 |

## 권장 사양

- **포맷**: JPG (배경이 단색이면), PNG (투명 배경 필요 시), WebP (파일 크기 최적화)
- **해상도**: 가로 최소 1000px 이상 (Retina 대응 시 1600px 권장)
- **파일 크기**: 200KB 이하 (WebP로 변환하면 더 작게 가능)
- **종횡비**: 캐러셀은 16:10 (자동 fit) — 원본 비율 그대로 저장하면 됩니다

## 파일이 없을 때

`onerror` 핸들러가 자동으로 "이미지 준비 중" 대체 UI를 표시합니다. 
파일을 저장하면 즉시 반영됩니다 (코드 수정 불필요).
